# Snippet Palace

O Snippet Palace é uma aplicação feita 100% com Inteligência Artificial.

O intuito do aplicativo é organizar e facilitar a reutilização de trechos de código. A ideia surgiu de uma necessidade pessoal trabalhando como engenheiro de dados, função na qual eu precisava utilizar várias queries de forma constante.

Faço o melhor para garantir a segurança e o funcionamento do aplicativo; contudo, não escrevi uma única linha de código e apenas utilizei a IA para realizar a sua construção. A única coisa que não foi feita por IA é este texto que você está lendo.


> **📥 [Baixe a última versão aqui](../../releases/latest)** - Windows 10/11 (x64)

## Por que usar o Snippet Palace? 

Cansado de procurar aquele snippet de código que você sempre esquece? O Snippet Palace é a solução perfeita para desenvolvedores que querem:

- Organizar código de forma profissional
- Encontrar snippets em segundos
- Aumentar produtividade no dia a dia
- Ter uma biblioteca pessoal de código sempre à mão

## Principais Funcionalidades

### Organização Poderosa
- ***Organização***: Organize todos os snippets do jeito que achar melhor, seja por pastas, projetos ou por linguagens de programação.
- ***Copia e Cola RÁPIDO***: Basta clicar duas vezes em um snippet para que ele seja copiado para a sua área de transferência.
- **Monaco Editor**: O mesmo editor do VS Code integrado.
- **Syntax highlighting**: Suporte para dezenas de linguagens
- **Tema escuro/claro**: Interface adaptável às suas preferências

## Features que eu acho legal
- ***Hot Snippers***: Salve snippets para serem acessados sem a necessidade de abrir o aplicativo.
- ***Busca rápida***: Busca inteligente de snippets com base no código e na descrição.
- ***JIRA Tickets***: Como tenho muitas queries que podem ser utilizadas para resolver chamados de dados, utilizo essa função para me lembrar de quando posso reutilizá-las.

### Produtividade
- **Atalhos de teclado**: Ctrl+N para criar, Ctrl+F para buscar e mais
- **Menu contextual**: Clique direito para ações rápidas
- **Onboarding interativo**: Tutorial guiado para novos usuários
- **Duplicação de snippets**: Clone e modifique rapidamente

## 📥 Como Instalar

### Opção 1: Instalador (Recomendado)
1. Acesse [Releases](../../releases/latest)
2. Baixe `Snippet Palace Setup x.x.x.exe`
3. Execute o instalador
4. Pronto! O app estará no menu Iniciar

### Opção 2: Versão Portátil
1. Acesse [Releases](../../releases/latest)
2. Baixe `Snippet Palace x.x.x Portable.exe`
3. Execute direto, sem instalação
4. Perfeito para uso em pen drives ou ambientes restritos

## Privacidade

Todos os seus dados são salvos **localmente** no seu computador (SQLite). Nenhuma informação é enviada para servidores externos.
